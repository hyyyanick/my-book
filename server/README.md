# my-book

## Project setup
```
npm install
cd server
npm install
```

### Run server
```
cd server
npm start
```
- run at `http://localhost:8080`
- GET : `http://localhost:8080/items`
- GET : `http://localhost:8080/items/:id`
- POST : `http://localhost:8080/items`
- PUT : `http://localhost:8080/items/:id`
- DELETE : `http://localhost:8080/items/:id`
